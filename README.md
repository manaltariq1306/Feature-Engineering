# Feature Engineering – Kaggle Course

This repository contains my work completed for the Feature Engineering course on Kaggle, assigned during the 6th Semester of my Machine Learning subject.

## Course Description

The course focuses on understanding how to prepare and improve data before training machine learning models. It explains how creating and selecting the right features can improve model performance.

## Topics Covered

- Introduction to Feature Engineering  
- Mutual Information  
- Creating New Features  

## Tools Used

- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Kaggle Notebooks  

## Learning Outcome

Through this course, I learned how to transform raw data into useful features and apply preprocessing techniques to improve machine learning model accuracy.
